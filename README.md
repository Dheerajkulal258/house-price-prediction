# House Price Prediction using Machine Learning

This project is a Machine Learning web application that predicts house prices based on:

* Square Feet
* Number of Bedrooms
* Number of Bathrooms

The application is built using **Linear Regression** and provides an interactive web interface where users can enter house details and get predicted prices instantly.

---

## 🚀 Features

* Predicts house prices using Machine Learning
* User-friendly web interface
* Real-time prediction output
* Flask-based backend
* Responsive design

---

## 🛠 Technologies Used

* Python
* Flask
* Scikit-learn
* Pandas
* NumPy
* HTML
* CSS

---

## 📊 Machine Learning Algorithm

* Linear Regression

The model is trained using housing data to learn the relationship between house features and prices.

## ▶️ How to Run the Project

### 1️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

### 2️⃣ Train the Model

```bash
python model.py
```

### 3️⃣ Run Flask Application

```bash
python app.py
```
